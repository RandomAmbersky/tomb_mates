# tomb_mates

Процесс написания игры можно посмотреть на [YouTube](https://www.youtube.com/watch?v=jMqC_VUEAgs)

![Скриншот](./screenshot.png 'Скриншотес')

## Запустить сервер

```
cd server
go run *go
```

## Запустить игру

###### Не будет работать без запущенного сервера

```
go run main.go
```
